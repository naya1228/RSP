# RSP - 게임 기획서

## 개요

| 항목 | 내용 |
|------|------|
| 장르 | 2D 턴제 온라인 대전 |
| 플랫폼 | Godot 4 데스크톱 / 모바일 |
| 인원 | 2인 (싱글플레이 AI 대전 지원) |
| 네트워크 | Ably WebSocket (멀티플레이 미구현) |

---

## 핵심 컨셉

10칸 징검다리 위에서 두 플레이어가 마주보고 전진하다가, 만나면 주머니에서 패를 동시에 꺼내는 권총결투 방식의 가위바위보 대결.

---

## 게임 흐름

```
1. 각자 출발점에서 시작 (A: 0번 칸, B: 9번 칸)
2. 턴마다 이동 선택 (전진/후진)
3. 같은 칸에서 마주치면 결투
4. 결투 승패 누적 → 3결투 2:1 조건 충족 시 강화패 픽
5. 상대 출발점 밟기 or 상대 MaxLoseStreak 연패 시 승리
```

---

## 맵

- 총 10칸 징검다리 (인덱스 0 ~ 9)
- 0번: A 출발점, 9번: B 출발점

```
[A출발:0] 1 2 3 4 5 6 7 8 [B출발:9]
```

---

## 이동 규칙

| 조건 | 이동 |
|------|------|
| 첫 턴 | 전진 1칸 또는 2칸 선택 |
| 이후 매 턴 | 전진 1칸 또는 후진 1칸 선택 |

- A는 오른쪽(+방향), B는 왼쪽(-방향)이 전진
- 결투에서 진 쪽이 다음 이동 턴을 가짐

---

## 패 시스템

### 초기 구성 (4장)
- 바위 1 / 보 1 / 가위 1 (보장) + 나머지 덱에서 랜덤 1장
- 덱에 남은 2장은 보충 시 사용

### 소모 방식
- 결투에서 낸 패는 손패에서 제거
- 손패가 0장이면 결투 불가

### 보충 시스템
- **보충 카운터**: 이동 턴 +1, 결투 승패 +1 (비김 제외)
- 카운터가 3 이상이면 덱에서 1장 드로우, 카운터 리셋
- **1회차 보충**: 무조건 강화패로 변환
- **2회차+ 보충**: 70% 강화패, 30% 일반패

---

## 강화패 시스템

### 획득 조건
- 비김을 제외한 3결투에서 2:1의 성적을 올린 플레이어가 강화패 1장 픽
- 픽 후 결투 카운터 리셋

### 강화패 종류 및 효과

| 패 | 이름 | 효과 |
|----|------|------|
| ★바위 | 강화 바위 | 상대 MaxLoseStreak -1 (최소 1) |
| ★보 | 강화 보 | 상대의 다음 결투를 블라인드로 만듦 |
| ★가위 | 강화 가위 | 가위vs가위 비김 시 강화 가위 쪽이 승리 |

### 블라인드 결투
- 강화 보에 맞은 플레이어는 다음 결투에서 자신의 패 종류가 숨겨짐
- 패 개수는 보이지만 어떤 패인지 알 수 없는 상태로 선택

---

## 결투 규칙

- 같은 칸에 두 플레이어가 위치하면 결투 발생
- 양쪽이 동시에 패를 선택 후 공개 (동시 공개로 치팅 방지)
- AI 싱글플레이: 커밋-리빌 없이 즉시 처리 (멀티플레이 시 구현 예정)

| 결과 | 효과 |
|------|------|
| 승리 | 진 쪽이 1칸 후퇴 |
| 패배 | 1칸 후퇴 + LoseStreak +1 |
| 비김 | 패만 소모, 재결투 (LoseStreak 유지, 드로우 카운터 미증가) |

---

## 탈락 / 승리 조건

| 조건 | 결과 |
|------|------|
| LoseStreak ≥ MaxLoseStreak | 즉시 탈락 |
| 상대 출발점 밟기 | 승리 |

- **MaxLoseStreak 기본값**: 3 (강화 바위에 맞으면 -1, 최소 1)
- **LoseStreak 리셋**: 결투에서 이기면 즉시 0, 비기면 유지

---

## 전략 요소

- **정보전**: 낸 패가 소모되므로 상대 남은 패 수를 추리 가능 (종류별 수량은 비공개)
- **심리전**: 강화 보로 블라인드 상태 만들기, 강화 바위로 상대 탈락 조건 낮추기
- **강화 관리**: 3결투 2:1 조건 노려서 강화패 획득
- **위치 전략**: 후진으로 결투 타이밍 조율, 상대 출발점 밟기 기회 포착
