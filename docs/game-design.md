# RSP - 게임 기획서

## 개요

| 항목 | 내용 |
|------|------|
| 장르 | 2D 턴제 온라인 대전 |
| 플랫폼 | Unity WebGL |
| 인원 | 2인 온라인 |
| 네트워크 | Photon PUN2 |

---

## 핵심 컨셉

10칸 징검다리 위에서 두 플레이어가 마주보고 전진하다가, 만나면 주머니에서 패를 동시에 꺼내는 권총결투 방식의 가위바위보 대결.

---

## 게임 흐름

```
1. 선/후 결정 (가위바위보)
2. 각자 출발점에서 시작
3. 턴마다 이동
4. 같은 칸에서 마주치면 결투
5. 상대 출발점 밟기 or 상대 3연패 시 승리
```

---

## 맵

- 총 10칸 징검다리
- 양 끝은 각 플레이어 출발점 (육지)
- 효과 칸: 양쪽 출발점에서 3칸 거리 (내용 미정)

```
[A출발] 1 2 [A효과] 4 5 6 [B효과] 8 9 [B출발]
```

---

## 이동 규칙

| 조건 | 이동 |
|------|------|
| 첫 턴 | 2칸 전진 |
| 이후 매 턴 | 전진 or 후진 1칸 선택 |

---

## 패 시스템

### 초기 구성
- 가위 2 / 바위 2 / 종이 2
- 카드가 아닌 선택지별 잔여 횟수 방식

### 소모 방식
- 선택한 패의 횟수 -1
- 횟수가 0이면 해당 패 선택 불가

### 보충
- 3턴마다 랜덤 패 1장 추가

---

## 결투 규칙

- 같은 칸에 두 플레이어가 위치하면 결투 발생
- 양쪽이 동시에 패를 선택 후 공개 (동시 공개로 치팅 방지)

| 결과 | 효과 |
|------|------|
| 승리 | 진 쪽이 1칸 후퇴 |
| 패배 | 1칸 후퇴 + 연패 카운트 +1 |
| 비김 | 패만 소모, 재결투 |

---

## 탈락 / 승리 조건

| 조건 | 결과 |
|------|------|
| 3연패 | 즉시 탈락 (물에 빠짐) |
| 상대 출발점 밟기 | 승리 |

- 연패 카운트는 이기면 즉시 0으로 리셋
- 비기면 카운트 유지

---

## 전략 요소

- **정보전**: 낸 패가 소모되므로 상대 남은 패를 추리 가능
- **심리전**: 같은 패 반복 사용 시 상대 패 구성 노출, 역이용 가능
- **위치 전략**: 후진으로 효과 칸 노리기, 결투 위치 조절
- **이동 선택**: 전진/후진 선택으로 결투 타이밍 조율
